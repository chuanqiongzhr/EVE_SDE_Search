# EVE SDE Search Tool

**EVE SDE Search Tool** 是一款轻量级的 EVE Online 静态数据（SDE）检索工具。无需配置数据库，开箱即用，支持快速搜索本地 JSONL 数据并查看详细属性。

---

## ✨ 核心功能

- **智能搜索**：支持中英文模糊匹配，多线程极速响应。
- **数据详情**：双击条目即可查看完整属性，提供清晰的树形视图，支持右键复制键值。
- **快速更新**：一键检查并下载最新的官方 SDE 数据包。
- **变更追踪**：自动下载并解析官方变更日志，直观展示版本变动内容。

## � 快速开始

1. **运行**：直接打开 `EVE_SDE_Search.exe`。
2. **初始化**：首次使用建议点击右上角 **“检查更新”** 下载最新数据。
3. **搜索**：输入关键词回车即可。
4. **查看**：双击结果查看详情，右键可快速复制 ID/名称。

## � 运行要求

- Windows 10/11 (64位)
- 无需安装任何运行库，解压即用。
